About This Project

Income Prediction with Fairness-Aware Machine Learning
Project Overview

This project demonstrates the application of fairness-aware machine learning techniques to income prediction. It uses the Adult Income Dataset from the UCI Machine Learning Repository to predict whether an individual's income exceeds $50,000 per year based on census data.

Fairness in Machine Learning

Machine learning models can inadvertently perpetuate or amplify biases present in training data. This project implements fairness-aware algorithms that aim to mitigate bias with respect to sensitive attributes such as gender and race, while maintaining predictive performance.

Fairness Metrics

The project evaluates models using the following fairness metrics:

  • Demographic Parity Difference: Measures the difference in positive prediction rates between privileged and unprivileged groups.
  • Equalized Odds Difference: Measures the difference in true positive and false positive rates between privileged and unprivileged groups.
Implemented Models

The project implements several machine learning models:

  • Logistic Regression: A baseline linear model for binary classification.
  • Random Forest: An ensemble learning method that builds multiple decision trees.
  • Fair Demographic Parity: A model constrained to satisfy demographic parity.
  • Fair Equalized Odds: A model constrained to satisfy equalized odds.
Dataset Information

Name: Adult Income Dataset
Source: UCI Machine Learning Repository
Instances: 48,842
Features: 14
Task: Binary Classification

The dataset contains census data extracted from the 1994 Census database. The prediction task is to determine whether a person makes over $50,000 a year.
